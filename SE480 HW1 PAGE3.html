SOME HTML TAGS

TAG DEFINITION
"!DOCTYPE html" The DOCTYPE html declaration defines that this document is an HTML5 document
"head" head tag contains meta information about the HTML page
"body" With body tag you can design the body part of the website
"p" With p tag you can write a paragraph
"title" With title tag you can change the name of the website.
"html" html is the root tag of the HTML page
"link" The link element defines the relationship between the current document and an external resource.
"div" The div tag is used as a container for HTML elements
